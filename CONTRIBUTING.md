# Contributing to Tab402

Thanks for your interest in Tab402. This is a small project and contributions are welcome.

## Getting started

1. Fork and clone the repo.
2. Install dependencies: `npm install`.
3. Copy `.env.example` to `.env` and fill in the values (see the README for setup).
4. Run the type checker before pushing: `npm run typecheck`.

## Guidelines

- Keep changes focused and match the style of the surrounding code.
- Run `npm run typecheck` and make sure it passes.
- If you change runtime behavior, describe how you tested it in the pull request.
- Do not commit secrets, private keys, or API keys.

## Reporting issues

Open an issue on GitHub with clear steps to reproduce. For security-sensitive reports, please contact the maintainer directly rather than opening a public issue.
